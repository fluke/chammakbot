NAME
gcloud sql backups - provide commands for working with backups of Cloud SQL instances
SYNOPSIS
gcloud sql backups COMMAND [--instance INSTANCE, -i INSTANCE] [GLOBAL-FLAG â€¦]
DESCRIPTION
Provide commands for working with backups of Cloud SQL instances including listing and getting information about backups for a Cloud SQL instance.
FLAGS
--instance   INSTANCE,   -i   INSTANCE
Cloud SQL instance ID.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
COMMANDS
COMMAND is one of the following:
describe
Retrieves information about a backup.
list
Lists all backups associated with a given instance.
NOTES
This command is in the Google Cloud SDK sql component. See installing components if it is not installed.
