NAME
gcloud alpha bigquery tables add-rows - adds records specified in a file to a specified existing table
SYNOPSIS
gcloud alpha bigquery tables add-rows TARGET_TABLE [--file FILE] [--fingerprint-job-id] [GLOBAL-FLAG â€¦]
DESCRIPTION
(ALPHA) The records are specified in the file as newline delimited JSON.
POSITIONAL ARGUMENTS
TARGET_TABLE
The table into which the rows are to be inserted.
FLAGS
--file FILE
The file containing the newline-delimited JSON. Defaults to stdin.
GROUP FLAGS
--fingerprint-job-id
Whether to use a job id that is derived from a fingerprint of the job configuration.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
NOTES
This command is in the Google Cloud SDK bigquery component. See installing components if it is not installed.
This command is currently in ALPHA and may change without notice.
