NAME
gcloud dns record-sets transaction add - append a record-set addition to the transaction
SYNOPSIS
gcloud dns record-sets transaction add DATA [DATA …] --name NAME [--transaction-file TRANSACTION_FILE; default="transaction.yaml"] --ttl TTL --type TYPE [--zone ZONE, -z ZONE] [GLOBAL-FLAG …]
DESCRIPTION
This command appends a record-set addition to the transaction.
POSITIONAL ARGUMENTS
DATA   [DATA   …]
DNS name of the record-set to be added.
FLAGS
--name NAME
DNS name of the record-set to be added.
--ttl TTL
TTL for the record-set to be added.
--type TYPE
Type of the record-set to be added.
GROUP FLAGS
--transaction-file TRANSACTION_FILE; default="transaction.yaml"
Path of the file which contains the transaction.
--zone ZONE, -z ZONE
Name of the managed-zone whose record-sets you want to manage.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
EXAMPLES
To add an A record, run:
$ gcloud dns record-sets transaction add -z MANAGED_ZONE \
    --name my.domain. --ttl 1234 --type A "1.2.3.4"
To add a TXT record with multiple data values, run:
$ gcloud dns record-sets transaction add -z MANAGED_ZONE \
    --name my.domain. --ttl 2345 --type TXT "Hello world" "Bye \
    world"
NOTES
This command is in the Google Cloud SDK dns component. See installing components if it is not installed.
