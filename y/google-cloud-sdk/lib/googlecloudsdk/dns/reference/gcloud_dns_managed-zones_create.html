NAME
gcloud dns managed-zones create - create a Cloud DNS managed-zone
SYNOPSIS
gcloud dns managed-zones create ZONE_NAME --description DESCRIPTION --dns-name DNS_NAME [GLOBAL-FLAG â€¦]
DESCRIPTION
This command creates a Cloud DNS managed-zone.
POSITIONAL ARGUMENTS
ZONE_NAME
Name of the managed-zone to be created.
FLAGS
--description DESCRIPTION
Short description for the managed-zone.
--dns-name DNS_NAME
The DNS name suffix that will be managed with the created zone.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
EXAMPLES
To create a managed-zone, run:
$ gcloud dns managed-zones create my_zone --dns_name my.zone.com. \
    --description "My zone!"
NOTES
This command is in the Google Cloud SDK dns component. See installing components if it is not installed.
