NAME
gcloud alpha dataflow logs list-messages - retrieve the logs from a specific job using the GetMessages RPC
SYNOPSIS
gcloud alpha dataflow logs list-messages JOB_ID [--after AFTER] [--before BEFORE] [--importance IMPORTANCE; default="warning"] [GLOBAL-FLAG â€¦]
DESCRIPTION
(ALPHA) This is intended for short-term use and will be removed once the CLI based on Cloud Logging is available.
POSITIONAL ARGUMENTS
JOB_ID
The job ID to operate on.
FLAGS
--after AFTER
Only display messages logged after the given time.
--before BEFORE
Only display messages logged before the given time.
--importance IMPORTANCE; default="warning"
Minimum importance a message must have to be displayed
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
