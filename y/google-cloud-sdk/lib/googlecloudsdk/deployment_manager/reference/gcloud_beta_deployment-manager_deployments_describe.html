NAME
gcloud beta deployment-manager deployments describe - provide information about a deployment
SYNOPSIS
gcloud beta deployment-manager deployments describe DEPLOYMENT_NAME [GLOBAL-FLAG â€¦]
DESCRIPTION
(BETA) This command prints out all available details about a deployment.
POSITIONAL ARGUMENTS
DEPLOYMENT_NAME
Deployment name.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
EXAMPLES
To display information about a deployment, run:
$ gcloud beta deployment-manager deployments describe my-deployment
NOTES
This command is in the Google Cloud SDK gcloud component. See installing components if it is not installed.
This command is currently in BETA and may change without notice.
