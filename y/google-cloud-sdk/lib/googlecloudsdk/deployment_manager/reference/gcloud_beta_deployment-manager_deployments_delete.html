NAME
gcloud beta deployment-manager deployments delete - delete a deployment
SYNOPSIS
gcloud beta deployment-manager deployments delete DEPLOYMENT_NAME [DEPLOYMENT_NAME …] [--async] [GLOBAL-FLAG …]
DESCRIPTION
(BETA) This command deletes a deployment and deletes all associated resources.
POSITIONAL ARGUMENTS
DEPLOYMENT_NAME   [DEPLOYMENT_NAME   …]
Deployment name.
FLAGS
--async
Return immediately and print information about the Operation in progress rather than waiting for the Operation to complete. (default=False)
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
EXAMPLES
To delete a deployment, run:
$ gcloud beta deployment-manager deployments delete my-deployment
To issue a delete command without waiting for the operation to complete, run:
$ gcloud beta deployment-manager deployments delete my-deployment \
    --async
To delete several deployments, run:
$ gcloud beta deployment-manager deployments delete deployment-one \
    deployment-two deployment-three
To disable the confirmation prompt on delete, run:
$ gcloud beta deployment-manager deployments delete my-deployment \
    -q
NOTES
This command is in the Google Cloud SDK gcloud component. See installing components if it is not installed.
This command is currently in BETA and may change without notice.
