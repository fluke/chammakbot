NAME
gcloud beta deployment-manager deployments cancel - cancel a pending or running deployment preview, update or creation
SYNOPSIS
gcloud beta deployment-manager deployments cancel DEPLOYMENT_NAME [--async] [GLOBAL-FLAG â€¦]
DESCRIPTION
(BETA) This command will cancel a currently running or pending operation on a deployment.
POSITIONAL ARGUMENTS
DEPLOYMENT_NAME
Deployment name.
FLAGS
--async
Return immediately and print information about the Operation in progress rather than waiting for the Operation to complete. (default=False)
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
EXAMPLES
To cancel a running operation on a deployment, run:
$ gcloud beta deployment-manager deployments cancel my-deployment
To issue a cancel command without waiting for the operation to complete, run:
$ gcloud beta deployment-manager deployments cancel my-deployment \
    --async
NOTES
This command is in the Google Cloud SDK gcloud component. See installing components if it is not installed.
This command is currently in BETA and may change without notice.
