NAME
gcloud compute regions - list Google Compute Engine regions

DESCRIPTION
List Google Compute Engine regions.
GLOBAL FLAGS

COMMANDS
COMMAND is one of the following:
describe
Describe a Google Compute Engine region.
list
List Google Compute Engine regions.
NOTES
