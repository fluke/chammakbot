NAME
gcloud compute project-info remove-metadata - remove project-wide metadata entries
SYNOPSIS
gcloud compute project-info remove-metadata [--all | --keys KEY,[KEY,…]] [GLOBAL-FLAG …]
DESCRIPTION
gcloud compute project-info remove-metadata can be used to remove project-wide metadata entries.
FLAGS
--all
If provided, all metadata entries are removed.
--keys KEY,[KEY,…]
The keys of the entries to remove.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
NOTES
This command is in the Google Cloud SDK compute component. See installing components if it is not installed.
