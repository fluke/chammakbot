NAME
gcloud compute backend-services edit - modify backend services
SYNOPSIS
gcloud compute backend-services edit NAME [GLOBAL-FLAG â€¦]
DESCRIPTION
gcloud compute backend-services edit can be used to modify a backend service. The backend service resource is fetched from the server and presented in a text editor. After the file is saved and closed, this command will update the resource. Only fields that can be modified are displayed in the editor.
The editor used to modify the resource is chosen by inspecting the EDITOR environment variable.
POSITIONAL ARGUMENTS
NAME
The name of the backend service to modify.
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
NOTES
This command is in the Google Cloud SDK compute component. See installing components if it is not installed.
