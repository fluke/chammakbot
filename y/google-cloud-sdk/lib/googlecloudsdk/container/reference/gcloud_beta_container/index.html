NAME
gcloud beta container - deploy and manage clusters of machines for running containers
SYNOPSIS
gcloud beta container GROUP | COMMAND [--zone ZONE, -z ZONE] [GLOBAL-FLAG â€¦]
DESCRIPTION
(BETA) Deploy and manage clusters of machines for running containers.
FLAGS
--zone   ZONE,   -z   ZONE
The compute zone (e.g. us-central1-a) for the cluster
GLOBAL FLAGS
Run $ gcloud help for a description of flags available to all commands.
GROUPS
GROUP is one of the following:
clusters
(BETA) Deploy and teardown Google Container Engine clusters.
operations
(BETA) Get and list operations for Google Container Engine clusters.
COMMANDS
COMMAND is one of the following:
get-credentials
(BETA) Fetch credentials for a running cluster.
NOTES
This command is in the Google Cloud SDK gcloud component. See installing components if it is not installed.
This command is currently in BETA and may change without notice.
